# LDMania

LDMania is a game where you play short mingames, round after round, while the difficulty scales over time, while trying to reach a score of infinity (yes, the score does reach infinity, that's just around e^710 actually, but who cares)

LDMania is developed by LDM Studio, initially for the SKG Official Game Jam 2026, in a period from the 17/07/26 to 24/07/26 (DD/MM/YY for y'all americans... not that there is a 17th or 24th month.. but you never know)

> D and M worked on the game while L didn't


The game is published on itch.io (LINK TO BE UPDATED)
